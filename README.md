# Rock, Paper, Scissors Game

A simple **Rock, Paper, Scissors** game built with **HTML, CSS, and JavaScript**, where you play against the computer.  
The game shows whether you **win**, **lose**, or **tie** each round and keeps track of your overall performance in a score table.

## How to Play
1. Open `Rock-Paper-Scissors.html` in your web browser.
2. Choose **Rock**, **Paper**, or **Scissors**.
3. The computer will make its choice automatically.
4. The result of the round (**win, lose, or tie**) is displayed on the screen.
5. The table updates automatically to show:
   - Total **Wins**
   - Total **Losses**
   - Total **Ties**
   - **Total games played**

## Features
- Play against the computer
- Real-time winner display
- Score table to track:
  - Wins
  - Losses
  - Ties
  - Total games
- Simple, responsive design

## Files
- `Rock-Paper-Scissors.html` – main game structure
- `Rock-Paper-Scissors.css` – styling for the game
- `Rock-Paper-Scissors.js` – game logic
- `images/` – images and icons used in the game
